The most frequent Tag

현재 150+ Tag 존재 but 25~30개 tag만 알면 충분하다

No.1 head - 제목에 해당하는 부분을 구분짓는 태그. 웹페이지의 정보를 담거나 외부 자료들을 해당 문서로 사용하기 위해 링크하는 태그를 담는 부모태그, title & meta 태그가 여기에 속한다.

No.2 body - 본문에 해당하는 부분을 구분짓는 태그.

No.3 html - 제작하는 html 문서의 시작과 끝을 나타내는 태그. 모든 내용을 담게 되는 부모 태그이다.

No.4 title - 웹페이지의 제목을 변경하는 태그. 검색엔진에서 검색이 되도록 한다.

No.5 meta - head 태그 안에 위치해야한다.문서 내용, 키워드, 누가 만들었는지 담고 있다.

No.6 div

No.7 a - 링크 태그

No.8 script

No.9 link

No.10 img - 이미지를 넣는 tag. 속성을 넣어준다. 순서 상관없음. ex) src=""를 적어준다.

No.11 p - 단락을 구분하는 tag, 의미로 단락을 구분짓고 싶을 때 유용

No.12 span

No.13 li - 리스트 목록 만드는 tag

No.14 ul - unordered list. li 태그의 부모 tag. 그룹핑 짓는 태그. ol(ordered list) - 목록의 내용의 자동 넘버링 해주는 태그

No.15 br - 줄바꿈 tag; 태그 수에 따라 줄간격 늘어남. 닫힘 tag 필요 X

No.16 style

No.17 h1

No.19 input

No.20 form

No.21 strong - 굵은 글씨(Bold체)로 강조하는 태그
